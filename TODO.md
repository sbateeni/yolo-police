# To-Do for Running ALPR Now

1. تأكد من تنصيب المتطلبات داخل البيئة الافتراضية:
   - `.venv\Scripts\python.exe -m pip install -r requirements.txt`
2. أنشأت مجلد التجربة:
   - `data/test_videos/`
3. حمل الفيديوهات المخصصة للتجربة باستخدام السكريبت:
   - `python download_test_videos.py`
4. تأكد من أن الفيديوهات غير مفقودة في المجلد ثم شغّل النظام:
   - `python main.py --video data/test_videos/<filename>.mp4`
   - أو افتح الواجهة: `python main.py --gui`
5. إذا كنت تريد تحميل مقاطع إضافية، أضف الروابط إلى `download_test_videos.py` ثم أعد تشغيل السكريبت.

## ملاحظات

- المجلد `data/test_videos/` تم إنشاؤه للاحتفاظ بالفيديوهات التجريبية.

- إذا لم يكن `yt-dlp` مُثبتًا، سيقوم السكريبت بطلب التثبيت تلقائيًا.

## تطوير صفحة جديدة لمراقبة البشر

1. إضافة صفحة جديدة في واجهة المستخدم تسمى "Human Analytics" أو "People Monitor".
2. استخدام كشف الأشخاص عبر YOLO أو موديل خاص بالبشر بدلاً من الاعتماد فقط على كشف المركبات.
3. إضافة عداد شامل:
   - عدد الأشخاص الكلي.
   - عدد الذكور.
   - عدد الإناث.
   - عدد الأشخاص الذين يرتدون ملابس داكنة / فاتحة / ألوان معينة إذا أمكن.
4. إضافة تصنيف سريع للباس على الوجوه أو الجسم (مثلاً: "معطف", "قميص", "فستان", "بدلة", "سترة") باستخدام نموذج ملابس أو تصنيف بسيط.
5. التقاط صورة وجه واحدة أو اثنتين لكل شخص مع تجنب التكرار:
   - حفظ أول صورة وجه واضحة لكل هوية تم اكتشافها.
   - إذا ظهر نفس الوجه لاحقًا، لا تُحفظ صورة جديدة.
   - احتفظ بأقل عدد من الصور الممكنة لأن الحذف لن يتم في البرنامج، وسيتولى المستخدم الحذف يدويًا.
6. بناء لوحة تحكم واسعة وشاملة تشمل:
   - مفتاح اختيار بين عرض المركبات وعرض الأشخاص.
   - أزرار بدء/إيقاف/إعادة تشغيل للكاميرا والفيديو.
   - زر تجريبي مباشر لتشغيل كاميرا EarthCam `bourbonstreet`.
   - عداد حي للأشخاص والذكور/الإناث والملابس.
   - قائمة سجل تظهر أحدث الوجوه والأشخاص مع الوقت والحالة.
   - لوحة تفاصيل لكل شخص تظهر:
     * النص المكتشف (مثل جنس الملابس أو نوعها)
     * صورة الوجه الملتقطة
     * حالة التكرار
     * إطار الظهور
7. تحديث الـ backend في `processor` لإدارة:
   - كشف الأشخاص.
   - تطابق الوجوه عبر الإطارات أو تتبع الهوية.
   - تخزين بيانات الوجوه بدون تكرار.
   - واجهة إخراج JSON/CSV للبيانات البشرية من دون صور الوجه.
8. التأكد من قابلية التوسيع لاحقًا لإضافة:
   - العدد الكلي مع أوقات الذروة.
   - تحذير عند وجود أكثر من عدد محدد من الأشخاص.
   - تقارير يومية/ساعة من صفحة الأشخاص.

## ✅ تم إنجاز صفحة الأشخاص بالكامل

### الملفات الجديدة:
1. **`processor/people_detector.py`** - كشف الأشخاص والوجوه والخصائص
   - استخدام YOLO لكشف الأشخاص (class 0)
   - استخراج الوجوه باستخدام Cascade Classifier
   - تقدير الجنس والملابس بناءً على الخصائص المرئية

2. **`processor/people_pipeline.py`** - معالجة الأشخاص والتتبع
   - عداد الأشخاص والذكور والإناث
   - منع تكرار الوجوه باستخدام MD5 hash
   - حفظ JSON/CSV للبيانات

3. **`ui/people_tab.py`** - تبويب عرض الأشخاص
   - عرض الإحصائيات الحية (ألوان ملونة)
   - قائمة سجل مع صور الوجوه
   - جدول تفاصيل شامل

### التحديثات:
- **`ui/workers.py`** - دعم وضعين (مركبات/أشخاص)
- **`ui/main_window.py`** - تبويبات وتبديل الأوضاع
- **`config.py`** - إضافة EarthCam URL للاختبار

## 🚀 خطة تطوير الميزات الأمنية المتقدمة (نظام الشرطة والأمن)

1. **التعرف على الأسلحة (Weapon Detection):**
   - دمج نموذج YOLO مخصص لاكتشاف الأسلحة النارية والبيضاء (Gun/Knife).
   - إطلاق إنذار فوري (تنبيه مرئي وصوتي بالواجهة) عند رصد سلاح.

2. **التعرف المتقدم على الوجوه والمطابقة (Face Recognition & Matching):**
   - استبدال نظام الجنس بنظام تعرّف شامل للوجوه يعتمد على `DeepFace` أو `FaceNet` للحصول على دقة عالية.
   - ربط النظام بقاعدة بيانات محلية لصور المطلوبين.
   - إطلاق إنذار عند التقاط صورة تتطابق مع شخص مطلوب.

3. **تحليل السلوك وكشف العنف (Violence & Suspicious Behavior Detection):**
   - استخدام نماذج `YOLO-Pose` أو `SlowFast` لفهم السلوك.
   - إرسال تنبيه في حالات العراك، السقوط، أو التسلق.

4. **تحديد نوع ولون السيارة بدقة (Vehicle Make/Model/Color):**
   - إضافة نموذج للتعرف على ماركة السيارة ولونها.
   - التحقق من توافق مواصفات السيارة مع اللوحة المقروءة لاكتشاف التزوير.

5. **التتبع الذكي عبر الكاميرات (Re-Identification / DeepSORT):**
   - دمج `DeepSORT` لإعطاء كل شخص هوية رقمية (ID) وتتبعه.
   - الاعتماد على الملابس والشكل العام للتعرف على الشخص إذا التفت أو اختفى وجهه.

6. **كشف الأجسام المتروكة (Abandoned Object Detection):**
   - تتبع الحقائب أو الأجسام الثابتة لفترة طويلة بدون صاحب لإطلاق تحذير.
